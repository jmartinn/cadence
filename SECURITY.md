# Security Policy

## Supported Scope

Cadence is a local-first iOS app. Security reports for the app code, bundled assets, repository configuration, and documentation are welcome.

## Reporting A Vulnerability

Use GitHub's private vulnerability reporting for this repository: open the repository's **Security** tab and choose **Report a vulnerability**.

Please include the affected area, reproduction steps, expected impact, and any suggested fix. Do not open a public issue for sensitive reports.

## Response

Reports are reviewed as maintainer availability allows. Confirmed issues will be handled privately until a fix or mitigation is ready.
